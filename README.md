# algorithms_practicum
Инструкция по запуску:
1)Скачиваем и устанавливаем Python на компьютер.
Скачать можно по ссылке https://www.python.org/downloads/
2) Скачиваем и устанавливаем интегрированную среду разработки (IDE),к примеру PyCharm(либо любую другую среду разработки.
Скачать Pycharm можно по ссылке https://www.jetbrains.com/pycharm/download/?ysclid=mpxx56zgrm442180999&section=windows
3)Запускаем среду разработки на своем компьютере,копируем код из выбранного файла и вставляем код в выбранной вами среду разработки.
4)Запускаете код.
5)Готово!
